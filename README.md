# 📊 ANALISIS-DEL-COMPORTAMIENTO-DEL-CLIENTE
Proyecto del Analisis del COmportameinto del Cliente. Se trabajo de forma transversal con Python, SQL, PowerBI, GAMMA

## 🚀 Descripción del Proyecto

Este proyecto presenta un **dashboard interactivo desarrollado en Power BI**, orientado al análisis del **comportamiento del cliente**, permitiendo visualizar de manera clara, dinámica y profesional indicadores clave relacionados con ventas, clientes, categorías de productos, edad, género y estado de suscripción.

El objetivo principal es facilitar la toma de decisiones mediante visualizaciones modernas, filtros interactivos y métricas estratégicas que permiten identificar patrones de compra, preferencias de los clientes y oportunidades de mejora comercial.

---

## 🎯 Objetivo

Analizar el comportamiento de los clientes a partir de datos comerciales, identificando:

- Cantidad total de clientes.
- Promedio del monto de ventas.
- Promedio de calificación del cliente.
- Distribución de clientes según estado de suscripción.
- Ventas y montos por categoría.
- Comportamiento por grupo de edad.
- Filtros dinámicos por sexo, categoría, tipo de envío y suscripción.

---

## 📌 Indicadores Principales

El dashboard incluye los siguientes KPIs:

| Indicador | Descripción |
|---|---|
| 👥 Número de clientes | Total de clientes registrados |
| 💰 Promedio de ventas | Promedio del monto de compra |
| ⭐ Calificación promedio | Nivel promedio de satisfacción del cliente |
| 📦 Categorías | Análisis de ventas por tipo de producto |
| 🧑‍🤝‍🧑 Grupo de edad | Comportamiento de compra según edad |
| 🔎 Filtros dinámicos | Segmentación por sexo, categoría, envío y suscripción |

---

## 🧩 Visualizaciones Incluidas

El reporte cuenta con visualizaciones diseñadas para una lectura rápida y efectiva:

- Tarjetas KPI con indicadores clave.
- Gráfico de dona para estado de suscripción.
- Gráficos de barras por categoría.
- Gráficos horizontales por grupo de edad.
- Segmentadores interactivos.
- Diseño visual personalizado con estilo moderno y colores corporativos.
